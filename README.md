# 🐾 Zoo Management Web Application

## 📌 Description
Application web développée en PHP permettant la gestion d’un zoo : animaux, personnel, soins et activités.  
Projet réalisé dans un contexte pédagogique pour mettre en pratique le développement web et la gestion de données.

---

## ⚙️ Fonctionnalités

- Gestion des animaux (ajout, modification, consultation)
- Gestion du personnel et des rôles
- Suivi des soins des animaux
- Système de parrainage
- Authentification des utilisateurs
- Gestion des données via base SQL

---

## 🛠️ Technologies

- PHP  
- HTML / CSS / JavaScript  
- SQL  
- Linux
